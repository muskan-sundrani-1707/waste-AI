About Waste-AI

Waste-AI is an intelligent web application designed to solve everyday waste management problems. It uses a PyTorch (ResNet50) model served via a Flask backend to provide instant, accurate waste classification (Organic, Recyclable, Hazardous, General) from a user's photo.

Why This Matters

Correct waste sorting is crucial for sustainability. It reduces landfill waste, improves the quality of recycling, and prevents hazardous materials from polluting our environment. This tool empowers users to make the right choice, every time.

How It Works (The Complete Flow)

  1. User Authentication: A user registers and logs in. The app uses a MySQL database to manage user data securely.
  2. Personalization: The user visits `home.html` and sees a greeting. In `settings.html`, they can set their locality (e.g., "Nehru Nagar").
  3. Schedule Check: The app fetches the user's area and displays the "Scheduled Collection" time from the database.
  4. AI Prediction: The user uploads an image to the `upload.html` page. The frontend sends this image and the `user_id` to the Flask `/predict` endpoint.
  5. Backend Analysis: The Flask server runs the PyTorch model, gets the prediction (e.g., "Recyclable"), and saves this result to the `predictions` table.
  6. Smart Advice: The frontend receives the "Recyclable" prediction and provides smart advice, including a dynamic Google Maps link to nearby "Recycling Centers".
  7. Track History: The user can visit `history.html` at any time to see a complete log of all their past classifications.
